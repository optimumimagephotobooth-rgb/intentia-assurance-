# Intake Questions

## Programme Identification

1. **Programme Name**
   - Official name of the training programme

2. **Programme Dates**
   - Start date: [Date]
   - End date: [Date]
   - Duration: [Hours/Days]

3. **Organisation**
   - Organisation name
   - Department/Division (if applicable)

4. **Participants**
   - Number of participants
   - Participant roles/levels (e.g., "Senior Leadership Team", "Middle Management")

## Training Intent

5. **Intended Purpose**
   - What was this programme intended to achieve?
   - What specific objectives were stated?

6. **Intended Outcomes**
   - What outcomes was the programme intended to produce?
   - What was it intended to change or improve?

## Explicit Exclusions

7. **What This Programme Was Not Intended To Do**
   - What was explicitly excluded from scope?
   - What was it not intended to guarantee or achieve?
   - What outcomes were explicitly not promised?

## Delivery Format

8. **Delivery Method**
   - In-person / Virtual / Hybrid
   - Format details (workshops, seminars, etc.)

9. **Delivery Details**
   - Number of sessions
   - Session duration
   - Location(s) (if applicable)

## Materials List

10. **Materials Provided**
    - List all materials provided:
      - Syllabi
      - Handouts
      - Presentations
      - Reading materials
      - Other documents

11. **Materials Format**
    - File formats and locations
    - Version numbers (if applicable)

## Notes

- No "why do you think" questions
- No open-ended reflection prompts
- Focus on factual documentation only
- All information must be confirmable by client
